## 0x00: Introduction
**#Rigorous** is a lightweight machine learning static library developed based on the **C++11** standard. **Now** it is open-source.
Including some classic ML algorithms: **BP-ANN**, C-**S**upport **V**ector **M**achine, and **R**andom **F**orest.

## 0x01: Getting Started
Since this project is **Header-only**, no complex compilation process or dynamic linking is required. You can simply move the source code directly to the **Include path** .

### Clone the Repository
```powershell
git clone "https://www.github.com/classzheng/rigorous.git"
```

### Integration
1. Copy all files under the `src/` directory to your project folder or Include path. 
2. Include the main entry header in your code:
   ```cpp
   #include "rigorous.h"
   ```
3. Ensure your compiler has C++11 or higher support enabled: 
   - GCC/Clang: `-std=c++11`
   - MSVC: `/std:c++11`
**CAUTION**: Including @classzheng, nobody ensures the program to work under **GCC** or **MSVC** as your exception. Hence, do as your willing.

## 0x02: Features Overview
*   ** Autograd (Automatic Differentiation)**: Supports BP automatic differentiation with a dynamic memory pool to manage operation nodes.
*   ** NeuralNetwork**: Implementation of BP-ANN supporting multiple hidden layers, including common activation functions and their derivatives.
*   ** C-SVM (Support Vector Machine)**:
    *   **BinarySVM**: Binary SVM based on the SMO (Sequential Minimal Optimization) algorithm.
    *   **MultipleSVM**: Supports multi-class classification tasks.
    *   **LSSVM**: Least Squares SVM, solved quickly via linear equation systems.
    *   **Kernel Trick**: Built-in with 7 kernel functions (Linear, Gaussian, Laplacian, Cauchy, Cosine, Polynomial, Triangular).
*   ** RandomForest**: *( Under Development...)*.

## 0x03: Callable API (Classes & Functions)
All functionalities are encapsulated under the `Rigorous` namespace.

### 1. Autograd System (`namespace BackwardGrad`)
*   `Autograd<T>`: Core data type containing `.data` (value) and `.grad` (gradient).
*   `Automem<T>`: Automatic memory management pool. 
*   **Common Functions**: `AgSin`, `AgCos`, `AgTanh`, and overloaded arithmetic operators.

### 2. Neural Network (`namespace NeuralNetwork`)
*   `UnitLayer<T>`: Definition of a single neural layer.
*   `NetworkUnion<T>`: Overall container for the neural network.
    *   `forward(input)`: Forward propagation. 
    *   `backward(target)`: Backward propagation to update weights. 
    *   `train(data_pair)`: Batch training. 

### 3. Support Vector Machine (`namespace SVMPackage`)
*   `Kernel<T>`: Configuration object for kernel functions. 
*   `BinarySVM<T>` / `MultipleSVM<T>`:
    *   `train(vx, ty, expects, limit)`: Model training.
    *   `predict(x)` / `decision(x)`: Perform prediction or output decision values. 
*   `LSSVM<T>`: `optimize()` for fast optimization. 

## 0x04:  Important Notes
*   **Memory Management**: When using the `Autograd` system, all temporary nodes generated during operations are managed by `Automem`. Ensure the `Automem` object's lifecycle covers your entire calculation process. 
*   **Module Toggles**: To speed up compilation, you can define the following macros before including `rigorous.h` to disable specific modules: 
    *   `#define $DISABLE_ANN` (Disable Neural Network)
    *   `#define $DISABLE_SVM` (Disable SVM)
    *   `#define $DISABLE_RF` (Disable Random Forest)
*   **Performance Limits**: As it uses STL containers and lacks SIMD optimizations (like AVX/SSE), performance may be limited when processing extremely large datasets.
